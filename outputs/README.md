# Food Price Inflation Prediction – India

This project forecasts India's Consumer Price Index (CPI) using a SARIMA time-series model.

## Overview
- Monthly CPI data (2013–2023)
- Time-series forecasting using SARIMA
- Future CPI prediction
- Streamlit web app deployment

## Tech Stack
- Python
- Pandas, Matplotlib
- Statsmodels (SARIMA)
- Streamlit
